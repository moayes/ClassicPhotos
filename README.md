ClassicPhotos
=============

A repository for the sample project of 'How To Use NSOperations and NSOperationQueues' tutorial.

In this tutorial, you’ll learn how to use concurrency in iOS applications! You’ll first create an app that doesn’t use multithreading at all, so it will appear very sluggish and unresponsive. Then you will rework the application to add concurrent operations.

http://www.raywenderlich.com/19788/how-to-use-nsoperations-and-nsoperationqueues
